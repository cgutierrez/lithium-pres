Routing API

Routing API

Continuation Routing

  • /example/action -> app\controllers\ExampleController::action
  • /admin/example/action -> app\controllers\admin\ExampleController::action

Routing API

Microframework Routing

  • “Controllers” simply have to be callable.
    • Controller classes are called using the new `__invoke` special method in PHP 5.3
  • Route matching based on environment conditions of the request. eg: HTTP verbs

Media Class

  • Registers handlers for various content types

Method Filters

      • Alter arguments
      • Modify return values
  • The Bad
    • It looks pretty ugly
    • Requires a reference to an object to filter non static methods
